现如今，人们越来越多地使用手机作为日常沟通、交流的第一选择，而在工作中，我们常常使用电脑🤔。这时，我们常常遇到这种情况——登录一个网站、输入自己的账号和密码、点击登录……“我们已发送一个验证码到您的手机上（省略）”这不由得令我感到繁杂以及不便😅。\
“有什么办法能让我直接粘贴验证码啊！😡”虽然验证码短短几个数字，但是 打开手机-解锁屏幕-在[**AirDMM-一个通知消息优化的解决方案**](https://github.com/sdjdll/AirDMM-)上查看验证码 这个过程，很容易打断我们工作的思路😥。\
于是，我创建了**Box**😄——这个项目并不是为了简简单单地做同步验证码这种功能，我有更大的野心——当在一个局域网中，两个设备（手机和电脑）同步剪贴板、同步消息、根据消息进入对应应用（例如：手机QQ弹出消息在电脑上点击后进入电脑QQ）甚至……更多🤗！\
\
希望是美好的，不过得落实在代码中🧐，我将会持续更新**Box**，希望它能成为你工作问题上的一个解决方案\
